# Roadmap

La v1 reste volontairement limitée: lire, rechercher, envoyer et récupérer explicitement une pièce jointe.

## Hors v1

- Marquer un message comme lu ou non lu.
- Déplacer un message.
- Supprimer, archiver ou vider un dossier.
- Fournir une administration web durable.
- Ajouter OAuth Gmail ou Microsoft.

## Évolutions possibles

- Gestion de plusieurs profils mail.
- Backend Linux Secret Service et Windows Credential Manager.
- Exemples de configuration pour davantage de clients MCP.
- Conteneur de démonstration avec serveur mail de test.
- Releases GitHub avec jar vérifiable.
- Confirmation renforcée pour les futures actions sensibles.

Les évolutions ne doivent pas réduire les limites de confidentialité: pas de secret dans les logs, pas de récupération massive sans borne, pas de pièce jointe automatique.
